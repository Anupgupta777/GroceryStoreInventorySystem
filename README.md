# Grocery Inventory System

A web-based Grocery Inventory Management System developed using ASP.NET Core MVC, Entity Framework Core, SQL Server, and Bootstrap.

## Features

* User Login & Logout
* Dashboard
* Product Management
* Category Management
* Supplier Management
* User Management
* Sales Management
* Responsive Admin Panel

## Technologies Used

* ASP.NET Core MVC
* C#
* SQL Server
* Entity Framework Core
* Bootstrap 5
